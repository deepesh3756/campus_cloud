# ☁️ CampusCloud Project
CampusCloud is a modern, web-based platform designed to streamline assignment management, submission, and evaluation processes for educational institutions. It provides a structured environment for Students, Faculty, and Administrators to interact efficiently with course materials and assessments.

## ✅ In-Scope Features (What we WILL deliver)

### A. User Management and Authentication

* Secure **login and logout** functionality.
* **Role-based access control** (RBAC) for Students, Faculty, and Admin. 
* Password reset and profile management capabilities.
* Session handling using **token-based authentication** (e.g., JWT).

### B. Student Module

* Student dashboard displaying assignment overview and notifications.
* Subject-wise assignment listing with status indicators (**Pending, Submitted, Evaluated**).
* Detailed assignment description view with attachments.
* File upload and submission using drag-and-drop or file browser.
* Tracking submission status and viewing grades and feedback.
* In-app notifications for new assignments and evaluation updates.

### C. Faculty Module

* Creation, modification, and deletion of assignments.
* Uploading assignment-related resources and documents.
* Viewing and managing student submissions.
* Manual evaluation of assignments.
* Publishing grades and feedback to students.
* Viewing student rosters for assigned subjects.
* Notifications for new submissions and system updates.

### D. Admin Module

* Management of student and faculty accounts.
* Subject and batch management.
* System-level configuration and settings.
* Broadcast announcements to all users.
* Maintenance of audit logs and activity tracking.

### E. Assignment Evaluation Support

* Basic **plagiarism detection** for submissions (specifically if the submission is text/not a file).
* **Zero Grade Policy:** Submissions made after the deadline will automatically result in a 0 grade.

### F. Notification System

* **In-app notifications** for assignment creation, submission, and evaluation events.
* Status indicators for unread notifications.

---

## ❌ Out-of-Scope Features (Exclusions)

The following functionalities are explicitly excluded from the scope of the current version of CampusCloud to ensure timely delivery and focus on core features. They may be considered for future phases.

* **Autonomus Evaluation:** Automatic grading or evaluation of assignments using AI/Machine Learning.
* **Internal Chat:** One-to-one or group chat functionality between students and faculty.
* **Attendance Management:** System for tracking or managing student attendance.
* **Quiz Module:** MCQ (Multiple Choice Question) based quiz/test assignments.
* **Mobile Application:** Development of native iOS or Android versions of CampusCloud.
* **SaaS Architecture:** Functionality to support multiple independent institutions (Multi-Institution SaaS app).
* **Localization:** Multi-language support (e.g., Hindi, Marathi, etc.).
